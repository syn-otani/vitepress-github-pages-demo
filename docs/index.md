## GitHub Pagesとは

GitHub Pagesは、GitHubリポジトリから直接ホストされる静的サイトホスティングサービスです。

### 主な特徴

| 特徴 | 説明 |
|------|------|
| **無料ホスティング** | Publicリポジトリなら無料で利用可能 |
| **カスタムドメイン** | 独自ドメインの設定が可能 |
| **HTTPS対応** | SSL証明書が自動で適用される |
| **CI/CD連携** | GitHub Actionsで自動デプロイ |

### 公開URL形式

- **ユーザーサイト**: `https://<username>.github.io/`
- **プロジェクトサイト**: `https://<username>.github.io/<repository>/`

## VitePressとは

VitePressは、Vue.jsをベースにした高速な静的サイトジェネレーターです。

### 主な特徴

- ⚡ **高速**: Viteによる高速なビルドとHMR
- 📝 **Markdown拡張**: Vue コンポーネントが使用可能
- 🎨 **テーマカスタマイズ**: Vue SFCでカスタマイズ可能
- 🔍 **検索機能**: ローカル検索を標準搭載
