## 共通仕様

### レイアウト

- ヘッダー: システム名、ハンバーガーメニュー
- 左メニュー: ナビゲーション（開閉可能）
- メインパネル: 検索条件 + 検索結果
- フッター: （空）

### 表示/編集モード

- トグルボタンで切り替え
- 編集モード時は編集可能セルが黄色背景
- 登録ボタンは編集モード時のみ表示
